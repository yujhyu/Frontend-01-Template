# 个人总结
>关于学习的方法

自己以往的学习方式并不能让知识形成体系化，我们需要使用整理法梳理知识之间的关系链接，再知识点冲突或疑问时使用追溯法获取更权威的解释，重新再整入知识体系中，这样知识体系才具完备性

>优秀的前端工程师是什么样的？
1. 领域知识完备并形成体系
2. 能力：编程能力、架构能力、业务敏感度、工程能力
3. 有自己明确的目标、职业规划

通过这两期的学习中我已经能探测到自己的不足点，今后需要补充的知识还有很多，需要通过不断的学习提炼知识体系。
